# Weather Forecast Management System 🌦️

## Overview

The Weather Forecast Management System is a web-based application that provides real-time weather information for different locations using a weather API. Users can search for any city and get details such as temperature, humidity, weather conditions, wind speed, and cloud coverage.

## Features

* Search weather information by city name.
* Display real-time temperature data.
* Show humidity and atmospheric conditions.
* Display wind speed and cloud coverage.
* User-friendly and responsive interface.
* Fetch live weather data using API integration.

## Technologies Used

* **HTML** - Structure of the web application.
* **CSS** - Styling and responsive design.
* **JavaScript** - Application logic and API integration.
* **OpenWeatherMap API** - Fetching real-time weather data.

## Installation and Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-forecast-management.git
```

2. Navigate to the project folder:

```bash
cd weather-forecast-management
```

3. Open `index.html` in your browser.

## How It Works

1. Enter the name of the city in the search box.
2. Click the search button.
3. The application sends a request to the weather API.
4. The API returns real-time weather data.
5. The application displays the weather details on the screen.

## API Used

This project uses the **OpenWeatherMap API** to retrieve live weather information.
